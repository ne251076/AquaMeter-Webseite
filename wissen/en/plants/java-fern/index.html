Java Fern (Microsorum pteropus)

Java fern is a classic midground plant in countless aquariums — robust, adaptable, and easy to propagate.

OriginSoutheast Asia
TypeEpiphyte (fern)
Lightlow to moderate light is sufficient
CO2not required
Growth ratemedium
Propagationrhizome division or plantlets on leaf tips
Difficultyeasy

Attach it right — don't bury it!

Like Anubias, java fern is an epiphyte — the rhizome must not be buried in the substrate, or rot can set in. Tie the plant to rocks or driftwood instead.

Propagation — unusually easy

Java fern has a practical advantage over many other epiphytes: small plantlets form on the leaf tips over time, which can simply be pulled off and tied to new rocks or driftwood — an easy way to get more plants without having to divide the rhizome.

Care and common problems

Older leaves can turn yellowish or brownish over time — that's normal and no cause for concern. Just gently trim these leaves off from the rhizome. Small black spots sometimes appear on leaves; these are harmless, but you can remove the affected leaf if it bothers you visually.
